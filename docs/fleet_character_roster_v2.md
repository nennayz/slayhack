# Fleet Character Roster v2

Purpose: revised character bible for NayzFreedom Fleet agents, PMs, and concept
crew. This version makes each character visually distinct enough for portraits,
dashboard cards, prompts, and future animation direction.

Naming note: Nayz is the Captain of the whole Fleet. Project PMs are powerful
operators, but they are not the Captain.

## Fleet Identity Rules

- Nayz is always the final Captain and owner identity.
- The crew should feel like sexy, tasteful, high-fashion pirate models aboard a
  premium operating fleet, not generic office avatars.
- Sexy means adult confidence, styling, posture, and fashion presence. It does
  not mean nudity, lingerie, explicit posing, or body-first framing.
- Each character needs a readable silhouette, hair direction, accent color, and
  unique tool.
- Each character needs a distinct visual heritage direction so the full team
  feels global, cinematic, and easy to tell apart.
- Active workflow characters can appear in live dashboard logic.
- Concept-only characters can appear in docs or planned cards, but should not
  imply live access to private or unmodeled workflows.

## Anti-Duplication Rules

- No two characters should share the same full hair formula: color plus length
  plus shape must differ.
- Blue hair is reserved for Mia's practical signal-scout silhouette, with a
  short wind-cut shape that reads fast, alert, and horizon-focused.
- Blonde is limited to clearly different uses: Slay has a platinum silver-blond
  wolf cut, and Genie has stage-performer waves. Roxy's final hair is pending
  redesign.
- Auburn hair is reserved for Robin's high command ponytail so she stays
  separate from Nami's seafoam-black waves and Bella's chestnut editorial waves.
- Black/dark hair must have different cuts: Nayz is long captain layers,
  Stadium is an athletic fade, Iris is an asymmetrical bob, and Lila is
  glass-straight with a violet underside.
- Every portrait should show the signature gear clearly enough that a small
  dashboard avatar still reads as that character.
- Fashion should remain ship-ready: belts, boots, naval tailoring, corset-like
  structure, jewelry, gloves, capes, harnesses, scarves, and deck-ready props.
- If two characters feel similar, change hair shape first, then prop, then color.

open a major new ship, or override brand-world direction.

## Visual Priority For Future Portraits

1. Silhouette must be recognizable before facial detail.
2. Hair color and shape must differ across the team.
3. Visual heritage direction should be visible through styling, face structure,
   hair texture, and fashion references without becoming costume stereotypes.
4. The signature object must appear clearly in every portrait or character card.
5. The accent color should be visible but not dominate the whole image.
6. Nayz and the eagle should feel mythic and central, while PMs feel powerful but
   below Captain authority.
7. Active workflow agents should look operational, not decorative.
8. Concept-only characters should remain beautiful but clearly future-bound.
9. Every character is styled as adult high-fashion pirate crew: confident,
   glamorous, and ship-ready, without explicit or voyeuristic framing.
